# Lampa Plugins & Balancers

Репозиторий кастомных плагинов и балансёров для [Lampa](https://lampa.io) / [Alcopac](https://alkopac.org).

## Плагины — `plugins/`

Плагины расширяют функционал Lampa (плееры, интеграции, UI).

| Плагин | Описание |
|--------|-----------|
| `external-player` | Открытие видео потоков во внешнем плеере (PotPlayer и др.) |

## Балансёры — `modules/`

Балансёры (модули-источники) для Alcopac — поиск и выдача ссылок на фильмы и сериалы.

| Модуль | Описание |
|--------|-----------|
| `zonafilm` | Источник [zonafilm.tv](https://zonafilm.tv). Поддерживает фильмы и сериалы, выбор озвучки прямо в списке серий, проксирование потоков. |

### ZonaFilm — подробнее

- **Поиск** — ищет по `title` и `original_title` на странице поиска zonafilm.tv.
- **Фильмы** — выдаёт список доступных переводов (озвучек) с выбором качества (360p–1080p).
- **Сериалы** — сезоны → серии. Для каждого сезона показываются **кнопки перевода** (`videos__button selector`) прямо над списком серий. Переключение перевода не меняет сезон.
- **Прокси** — потоки могут проксироваться через `/proxy` (настройка `proxyStreams` в `manifest.json`).
- **Кэш** — поиск, плейлисты и видео-инфо кэшируются для скорости.
