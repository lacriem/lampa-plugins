# Lampa Plugins & Balancers

Репозиторий кастомных плагинов и балансёров для [Lampa](https://lampa.io) / [Alcopac](https://alkopac.org).

## Плагины — `plugins/`

Плагины расширяют функционал Lampa (плееры, интеграции, UI).

| Плагин | Описание |
|--------|-----------|
| `external-player` | Открытие видео потоков во внешнем плеере (PotPlayer и др.) |

## Балансёры — `modules/`

Балансёры (модули-источники) для Alcopac — поиск и выдача ссылок на фильмы и сериалы.

| Модуль | Описание |
|--------|-----------|
| `zonafilm` | Поиск и просмотр на zonafilm.tv |
